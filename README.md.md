## Task 3 — Classic Calculator

### Description
A fully functional, beautifully styled calculator built using HTML, CSS, and JavaScript. The calculator supports all four basic arithmetic operations and includes additional features like percentage calculation, backspace/delete, decimal input, and full keyboard support — all wrapped in a premium dark-themed UI.

### Tech Used
- **HTML5** — Semantic button layout and display structure
- **CSS3** — CSS Grid for button alignment, gradient backgrounds, box shadows, hover and active state animations
- **JavaScript** — Event listeners, DOM manipulation, expression handling, keyboard input support
- **Google Fonts** — Poppins typeface

### Features Implemented
- Addition, Subtraction, Multiplication, Division
- Percentage (%) calculation
- Backspace (⌫) to delete last character
- AC (All Clear) to reset
- Decimal point with duplicate-dot prevention
- Expression display + Result display (two separate outputs)
- Operator-replacement logic (replaces last operator if pressed twice)
- **Full keyboard support** — numbers, operators, Enter, Backspace, Escape, and period
- Error handling for invalid expressions

### CSS Grid Usage
```css
.btn-row {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 14px;
}
```
The `0` button spans 2 columns using `grid-column: span 2` as required by the task.

### What I Learned
- Using CSS Grid to create a structured, proportional button layout
- JavaScript event listeners for both click and keyboard interactions
- Handling edge cases in calculator logic (e.g., consecutive operators, leading dots, division by zero)
- Building a polished, production-quality UI with gradients, shadows, and hover effects
- Separating expression tracking and result display for better UX

### Output
A fully working, keyboard-accessible calculator that handles all basic arithmetic operations with a clean, premium dark UI.
